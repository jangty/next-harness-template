description: 현재 diff 셀프리뷰 (타입/린트/불변식/정의완료)
---

현재 변경(diff)을 스스로 리뷰한다. 성공은 침묵, 문제만 자세히 보고한다.

체크 항목:
1. **타입** — `npm run typecheck` 통과?
2. **린트** — `npm run lint` 통과? 경계 린트 위반 없는지(의존성 방향).
3. **불변식** — `CLAUDE.md` 의 불변식 목록을 위반하지 않는가? 각 불변식의 강제 메커니즘이 실제로 걸려 있는가.
4. **정의 완료(Definition of Done)** — 테스트 추가/갱신됨? 관련 `docs/` 신선한가(드리프트 없음)? 새 결정이면 ADR 기록됨?
5. **비밀/산출물** — diff에 `.env*`·빌드 산출물이 섞이지 않았는가.

각 항목을 ✅/⚠️/❌ 로 요약하고, ❌·⚠️ 에는 구체적 위치(file:line)와 수정 지침을 붙인다.
